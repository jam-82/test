Introduction

JavaScript is a cross-platform, object oriented scripting language.

It is a small and lightweignt language. Inside a host environment (for example, a we browser).

  • JavaScript can be connected to the objects of its environment to provide programmatic control over them.

JavaScript and Java
JavaScript and Java are similar in some ways but fundamentally different

The JavaScript language resembles Java but does not have java's static typing and strong type checking.

JavaScript has a prototype-based object model.
  • The prototype-based model provides dynamic inheritance
  • JavaScript is a very free-form language compared to Java.
Hello world

To get started with writing JavaScript, open the Scratchpad and write your first "Hello world" JavaScript code:

function greetMe(yourName) { alert("Hello " + yourName); } greetMe("World");
Variables

A JavaScript identifier must start with a letter, underscore(_), or dollar sign ($); subsequent characters can also be digits (0-9).

JavaScript is case sensitive

Variable scope

When you declare a varialble outside of any function, it is called a global variable, because it is available to any other code in the current document

if (true) { var x = 5; } console.log(x); //5

This behaviour changes, when using the let declaration introduced in ECMAScript 2015

if (true) { let y = 5; } console.log(y); // ReferenceError: y is not defined
Constants

You can create a read -only,named constant with the const keyword.

  • The syntax of a constant identifier is teh same as for a variable identifier/

You cannot declare a constant with the same name as a function or variable in the same scope.

However, object attributes are not protected, so the following statement is executed without problems.

const MY_OBJECT = {"key": "value"}; MY_OBJECT.key = "otherValue";
if...else statement

Use the if statement to execute a statement if the logical condition is true.

if (condition) { statement_1; } else { statement_2; }
